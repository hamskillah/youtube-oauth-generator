# 🔑 YouTube OAuth TV Token Generator

> ⚠️ Not fully automated — manual phone approval required.

Generate multiple YouTube OAuth TV refresh tokens.

## 🚀 Quick Start
`git clone https://github.com/hamskillah/youtube-oauth-generator.git && cd youtube-oauth-generator && bash generate.sh`

## 📋 Requirements
bash, curl, python3, Smartphone

## 🔧 Steps
1. Get OAuth Client ID from Google Cloud Console (TV type)
2. Run `bash generate.sh`
3. Enter Client ID & Secret
4. Approve on phone: https://www.google.com/device
5. Token displayed

## ⚠️ Important
- **Publish your OAuth app** or tokens expire in 7 days
- One Client ID per Google account
- Use burner accounts for bots
